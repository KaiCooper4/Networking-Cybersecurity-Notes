**Caches Size/Type**
Temp data storage; L1: fastest, smallest, most expensive; L2: 2nd fastest; L3: slowest

Hyperthreading - Allows a single CPU to accept & process two independent sets of instructions.

Multicore - Contains two or more actual processing cores.

Architecture - 64-bit allows the CPU to process higher data types, x86 or x84; 32-bit is limited to 4 GB of data address space.

**Sockets**
Zif Socket: Zero Insertion Force; uses a lever to push spring contacts apart for easy insertion of the pins, and ten shifts the contacts to grip the pins.
PGA: Pin Grid Array; pins arranged in square shape; typically AMD
LGA Socket: Land Grid Array; pins are on the socket rather than the circuit; typically Intel
BGA: Ball Grid Array; mobile/laptop processor
Intel Sockets: 775, 1155, 1156, 1366
AMD Sockets: 940, AM2, AM2+, AM3, FM1, F
Heat sinks (passive) / fans (active) / thermal compound / liquid cooling
