**Methods**
GET Request - Get info from web server.
POST Request - Submit data to server & create new records.
PUT Request - Update info.
DELETE Request - Delete info/records.

**Status Codes**
100-199: Info Response
200-299: Success
300-399: Redirection
400-499: Client Errors
500-599: Server Errors
200 - OK
201 - Created
301 - Permanent Redirect
400 - Bad Request
401 - Not Authorized 
403 - Forbidden
404 - Page Not Found
405 - Method Not Allowed
500 - Internal Service Error
503 - Service Unavailable
