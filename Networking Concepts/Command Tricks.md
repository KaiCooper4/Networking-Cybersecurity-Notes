Reading a dashed filename: cat ./-
Reading a filename w/ spaces: cat "spaces in the filename"
Reading a hidden file within a directory: ls -a 

**Windows**
ipconfig /release - removes IP from computer
ipconfig /renew - rolls back IP
ipconfig /displaydns /flushdns /registerdns

/release -> /flushdns -> /renew -> /registerdns (common troubleshooting procedure)
